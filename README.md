# Bottled time

Add a Bottled time card to speed up some harvest and exploration process.
Can be stack together to allow greater effects.
Currently only work on:
- Berry Bush
- Apple Tree
- Banana Tree
- Coconut Tree
- Cotton Tree
- Sugar Cane
- Tree
- Lumber Camp
- Rock
- Quarry
- Sand Quarry
- Iron Deposit
- Iron Mine
- Gold Deposit
- Gold Mine
- Exploration Type card

# Recipe

1x Empty Bottle + 5x Sand = 1x Bottled Time
